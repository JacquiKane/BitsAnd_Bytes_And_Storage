Measuring Data Storage

Storage Units

Storage Units

A bit is the smallest unit of information, given by a 0 or a 1. A byte is a group of 8 bits.

210, 1024, approximates to 1000 or 103 and the prefix kilo is used for 1024, in reference to bit or byte count. Following from this, we use prefixes that approximate values in base 10, though the real quantities are measured in powers of 2.

quantity - power value approximation
kilobyte KB 210 1024 103
megabyte MB 220 1,048,576 106
gigabyte GB 230 1,073,741,824 109
terabyte TB 240 1,099,511,627,776 1012
petabyte PB 250 1, 125, 899, 906, 842, 624 1015
exabyte EB 260 152, 921, 504, 606, 846, 976 1018
zettabyte ZB 270 1,180,591,620,717,411,303,424 1021
yottabyte YB 280 1,208,925,819,614,629,174,706,176 1024

Bits are represented with a lower case b, using the same prefix. 1Kb is 1024 bits. 1 KB is 1024 bytes. To get the number of bits in a byte, multiply by 8.

Primary Storage

Primary Storage, Random Access Memory

Temporary (volatile) storage in the computer, stores information for immediate use during processing. Also referred to as primary storage or main memory. Contents clear when there is no power source.

The speed and performance of a system correlates to the amount of RAM installed. If the system has too little RAM, it can be slow. Too much RAM has no added benefit, because an operating system can only address up to a specific limit of memory resources, due to address bus and addressing schema constraints.

Make sure RAM is compatible with the other components in a computer system.

According to Windows 11 specifications, 4GB+ of memory is recommended. OS X works best with 8GB of RAM, and Linux distros range from a very low requirement of 512 MB, and upwards.

Secondary Storage

Disk Drive, Secondary Storage

Disks represent permanent storage in the computer system.

An HDD records data magnetically and has multiple moving parts - spinning magnetic disks and a read-write head. An SSD is similar to a flash drive. Both store data permanently.

The disk drive stores the operating system,and any installed apps, as well as the user files and folders.

Windows 11 takes up about 18 GB of space on disk, but MicroSoft recommends that 64 GB be available for a Windows 11 install, to cater for temporary files written to disk. Mac OS X (depending on version) requires aroung 35GB. Space requirement is considerably smaller for distros of Linux, although this varies.

Disk capacity is displayed in MB (megabytes), GB (gigabytes), or TB (terabytes).Most modern computer systems come with disk drive capacity upwards of 256 GB.
